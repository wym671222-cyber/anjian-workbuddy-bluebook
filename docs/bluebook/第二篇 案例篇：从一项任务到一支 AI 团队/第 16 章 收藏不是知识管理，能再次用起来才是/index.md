# 第 16 章 从收藏资料到可复用知识资产

收藏只是把资料放进一个地方；知识管理要让资料能被找到、理解、核验并再次用于任务。本章讲通用方法，不公开真实知识库目录或内部文档。

## 可应用到哪些任务

- 保存政策、行业文章、报告、视频笔记和培训资料。
- 为专题工作建立来源页、概念页和分析页。
- 从多份材料中提炼可复用方法、模板和问题清单。
- 为以后检索、写作和研判保留可靠出处。

## 任务目标

形成“原始素材不丢、每项知识有来源、重复内容可合并、不确定内容有标记、知识能够回到具体任务”的体系。

## 执行步骤

1. **保留素材层**：原始文件或链接保持不变，记录来源和时间。
2. **建立来源页**：每项材料记录核心主张、关键细节、可复用方法、个人评注和来源说明。
3. **提炼知识点**：一个知识点解决一个问题，并链接回来源页。
4. **建立主题入口**：用主题、任务和状态组织，不只依赖文件夹。
5. **处理不确定性**：无法核实的内容标记“待核验”，不进入正式结论。
6. **定期复用与更新**：记录知识被哪个任务使用，过期内容保留历史状态。

## 合成示例

将三篇公开文章整理为“会议待办提取方法”知识卡。每篇文章各有来源页，知识卡引用具体段落，并注明哪些步骤来自来源、哪些是整理者的建议。

### 可复制任务模板

```text
请把当前目录中的资料整理为可追溯知识条目。

主题：{主题}
使用目的：{以后要解决的任务}
材料范围：{文件或链接}

每项材料分别输出：
1. 核心主张；
2. 关键细节；
3. 可复用方法；
4. 待核验内容；
5. 我的评注占位；
6. 来源、日期和原文件位置。

不要合并成一个无法追溯的大摘要；不要修改原始材料；
重复知识先建立关联，等待确认后再合并。
```

## 人工确认点

- 来源是否真实、完整并允许保存。
- 摘要是否改变原意，评注是否与原文分开。
- 重复内容是否真的可以合并。
- 敏感材料是否只保留索引，不扩写正文。

## 输出物与验收标准

- 每项来源有独立记录和稳定标识。
- 知识点能够回到来源，不是无出处结论。
- 待核验、已确认、已过期状态清楚。
- 主题入口能回答“这项知识可用于什么任务”。

## 常见弯路与安全边界

- 大批资料压成一个摘要会失去出处，应一项来源一条记录。
- 只收藏不写使用目的，日后仍找不到。每条知识关联任务。
- 把宣传包装一起保留会稀释方法，应区分事实、观点和推广内容。
- 不公开内部知识目录、不复制敏感原文、不静默修改来源。可复用来源页、知识卡、主题索引和待核验清单。
