# 第二篇 通用办公实战：从任务到可验收成果

本篇围绕安居建业常见办公任务提供可采用的方法。正文只使用合成样例，不展示内部项目、系统、部门数据或未经确认的公司流程。

## 章节安排

| 章节 | 主题 | 先解决什么问题 |
|---|---|---|
| 第11章 | Word、Excel、PPT 办公交付与格式检查 | 让多种交付物事实一致、格式可验收 |
| 第12章 | 文件、版本、终稿与归档治理 | 保护原件并理清版本和目录 |
| 第13章 | 安全远程协作与故障排查 | 用最小权限查看、确认和停止任务 |
| 第14章 | 把模糊需求变成可验收任务 | 在开工前明确目标、输入和验收 |
| 第15章 | 信息收集、人工筛选、复核与发布 | 建立有人把关的信息处理链 |
| 第16章 | 从收藏资料到可复用知识资产 | 让知识有来源、能查找、可复用 |
| 第17章 | 结构化数据分析与可视化表达 | 让口径、证据和限制更清楚 |
| 第18章 | 公文编制 | 从事实和依据到可审核初稿 |
| 第19章 | 会议与待办 | 把结论转成可跟踪行动 |
| 第20章 | 制度与知识管理 | 管理多文档、版本和引用关系 |
| 第21章 | 项目研判分析 | 把资料、证据、缺口和风险组织清楚 |

## 开始方式

第一次进入本篇，建议先读第14章形成任务卡，再根据输出物选择其他章节。涉及外发、审批、删除、覆盖和专业判断时，必须保留人工确认。
