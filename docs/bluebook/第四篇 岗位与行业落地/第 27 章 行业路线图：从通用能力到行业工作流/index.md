# 第 27 章 行业路线图：从通用能力到行业工作流

行业方案不是把“生成周报、做 PPT”换一个行业名称。真正的行业化至少包含四层：行业对象、专业流程、系统与数据、责任和监管边界。

## 识别行业场景的五个问题

1. 工作对象是什么：订单、设备、病例、合同、课程还是内容素材？
2. 流程由哪些岗位和系统共同完成？
3. 哪些字段、口径或版本必须保持一致？
4. 错误会造成什么后果，谁拥有最终责任？
5. 哪一步最重复、最耗时，又最容易人工验收？

只有第五类步骤适合作为首批 AI 试点。

## 制造业

```mermaid
flowchart LR
    A[设备/生产/质检记录] --> B[归集与字段校验]
    B --> C[异常与趋势分析]
    C --> D[日报/周报/质量报告]
    D --> E[人工确认与闭环]
```

| 场景 | 输入 | 交付物 | 专业约束 |
|-|-|-|-|
| 生产报表 | ERP 导出、班组表、停机记录 | 产量、良率、停机原因 | 班次、物料、工序口径一致 |
| 质量记录 | 检验表、图片、标准规范 | 缺陷分类、异常批次、报告草稿 | 不代替质量放行 |
| 设备知识 | 手册、维修记录、故障代码 | 排障建议、备件清单 | 安全操作和停机权限 |
| 供应链周报 | 订单、库存、交期 | 风险清单、催办建议 | 数据时点和供应商权限 |

## 跨境电商

跨境电商的难点不是单一内容生成，而是多平台、多语言、多币种、多时区和多规则并存。

| 场景 | WorkBuddy 工作流 | 验收重点 |
|-|-|-|
| 商品上新 | 产品资料 → 卖点 → 多语种草稿 → 平台格式 | 参数、禁用词、商标和本地表达 |
| 客服与评价 | 工单聚类 → 回复草稿 → FAQ 更新 | 订单事实、退款承诺、语气 |
| 经营复盘 | 订单/广告/库存 → 统一口径 → 利润与异常 | 汇率、时区、归因窗口 |
| 供应链协同 | 采购、交期、物流 → 风险清单 | 单号、时间、责任人 |

## 金融、咨询与财税

这类行业的核心资产是证据、分析口径和专业判断。AI 适合缩短资料处理时间，不适合隐藏来源后直接给结论。

| 场景 | 基础版 | 进阶版 | 人工关口 |
|-|-|-|-|
| 行业研究 | 来源表和资料摘要 | 证据矩阵、定期跟踪 | 观点与投资含义 |
| 咨询项目 | 访谈纪要、材料盘点 | 假设树、分析底稿、报告 | 建议与客户承诺 |
| 财税材料 | 单据识别、字段核对 | 申报资料清单、差异检查 | 申报和税务判断 |
| 经营分析 | 多表汇总 | 指标解释、异常归因 | 会计口径和管理决策 |

## 医疗医药

医疗医药强调证据等级、版本控制、隐私和专业资质。首批场景应避开诊断和治疗决定。

| 场景 | 可由 AI 辅助 | 不应自动完成 |
|-|-|-|
| 文献整理 | 检索结果去重、摘要、证据表 | 临床结论和治疗建议 |
| 生产质量 | 批记录完整性检查、培训材料 | 批次放行和偏差定责 |
| 医学/市场材料 | 来源核对、禁用表述提示 | 宣传合规最终批准 |
| 内部知识 | 制度问答、版本定位 | 超权限访问患者或试验数据 |



## 文化传媒与广告

视频团队案例已经展示了多 Agent 内容流水线。行业落地时，还需要在创作效率之外加入版权、品牌和发布控制。

```mermaid
flowchart LR
    A[热点与客户 brief] --> B[选题评分]
    B --> C[脚本与分镜]
    C --> D[素材与制作]
    D --> E[事实/版权/品牌审核]
    E --> F[人工发布]
    F --> G[数据复盘]
```

可沉淀资产，品牌语气、禁用词、事实来源库、授权素材库、平台格式模板、选题评分表、脚本 Skill、分镜模板和复盘指标。

## 教育与培训

| 场景 | 输入 | 交付物 | 约束 |
|-|-|-|-|
| 企业培训 | 制度、课程、案例 | 学习路径、课件、题库 | 版本和适用人群 |
| 教学备课 | 课程标准、教材、学情 | 教案、活动、练习 | 知识准确和年龄适配 |
| 学习反馈 | 作业、评分规则 | 共性问题、反馈草稿 | 不用敏感特征贴标签 |
| 招生内容 | 批准的课程资料 | 页面和咨询话术草稿 | 不夸大效果和承诺 |

## 建筑装饰与科技服务

这类行业以项目为单位，材料多、版本多、责任链长。WorkBuddy 适合做资料盘点、标书初稿、合同差异和项目周报。

| 场景 | 交付物 | 风险控制 |
|-|-|-|
| 招投标 | 响应矩阵、材料清单、标书初稿 | 逐条对应招标文件，不虚构资质 |
| 方案设计 | 需求摘要、方案结构、汇报材料 | 专业计算和图纸需资质人员确认 |
| 项目管理 | 纪要、进度、风险和变更清单 | 版本、签字和责任主体可追溯 |
| 合同与报价 | 条款差异、报价表检查 | 金额、税率和承诺人工复核 |
