## 按任务责任选择入口

| 任务责任 | 建议组合 | 首要验收 |
|---|---|---|
| 材料起草 | 第14章 → 第11或18章 | 事实有依据、格式可交付 |
| 复核审批 | 第20章 → 附录D | 版本、权限、例外和风险清楚 |
| 会议组织 | 第14章 → 第19章 | 结论和待办可追踪 |
| 数据分析 | 第17章 → 第11章 | 口径可复算、图表有证据 |
| 项目经办 | 第12章 → 第21章 | 资料完整、问题有来源 |
| 知识维护 | 第16章 → 第20、23章 | 来源可追溯、方法可复用 |

## 高风险动作速查

| 动作 | 默认处理 |
|---|---|
| 删除、覆盖、批量移动 | 先预览，保留原件，明确确认 |
| 对外发送、公开发布 | 脱敏检查，确认接收范围，人工执行 |
| 审批、签发、盖章 | AI 只辅助准备，不代替有权限人员 |
| 投资、法律、财务、工程判断 | 形成证据与问题清单，交专业人员判断 |
| 批量写入业务系统 | 小样本、幂等保护、日志和回退 |
| 录音、转写、读取个人信息 | 先确认授权和最小使用范围 |
