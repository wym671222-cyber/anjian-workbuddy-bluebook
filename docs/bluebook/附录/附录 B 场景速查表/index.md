## 基础办公速查

| 我想做什么 | 推荐能力 | 人工检查点 |
|-|-|-|
| 安排、修改或取消腾讯会议 | 腾讯会议 Skill | 主题、时间、时区、参会人和取消范围 |
| 获取会议录制、转写和智能纪要 | 腾讯会议 Skill | 录制告知、访问权限、专有名词和数字 |
| 把会议内容整理成纪要 | 转写 + 会议纪要模板 | 事实、决定、建议和未决问题分开 |
| 把行动项写入任务系统 | 任务连接器 / API | 负责人、截止日期、验收物和重复写入 |
| 根据会议生成 PRD | PRD 模板、产品经理 Skill | 只使用确认需求，保留目标外和未决问题 |
| 整理邮件和生成回复草稿 | 邮件连接器、内容整理 | 发件人、对象、附件和发送确认 |
| 根据任务与会议生成日报周报 | 任务、日历、文档 Skill | 成果有任务或文件证据，不把计划写成完成 |
| 批量整理和重命名文件 | 文件整理、DOCX/PDF | 先预览和复制，不覆盖、不删除 |
| 合并 Excel 并对账 | Excel、数据分析 | 主键、口径、异常清单和金额守恒 |
| 比较两版制度、合同或方案 | 文档对比 | 金额、日期、责任、条件和否定表达 |
| 根据纪要生成汇报 PPT | PPT、图表、品牌模板 | 先大纲和证据映射，一页一个结论 |
| 把公司制度做成问答 | 知识库、搜索 | 回答带条款来源，未命中转人工 |

## 知识管理速查

| 我想做什么 | 建议入口 | 人工检查点 |
|-|-|-|
| 随口记录一个想法 | 灵感捕手、ima | 保留原话，不自动扩写成事实 |
| 微信文件进入知识库 | ima-skills | 目标库、权限、重复与隐私 |
| 微信收藏批量整理 | 微信收藏知识库 | 抓取失败、来源、版权与重复 |
| 整理 WPS 文档和碎片笔记 | WPS 知识库、kdocs skill | Key、测试目录、分享权限 |
| 建本地 Obsidian Wiki | LLM-Wiki 中文版 | Vault 备份、命名与批量写入 |
| 网页转 Obsidian 笔记 | url-to-obsidian | 来源、日期、版权和原始链接 |
| 会议沉淀为项目记忆 | 纪要 + 知识库 Skill | 事实、决策、推测和待办分开 |
| 把知识变成自媒体内容 | 证据包 + 内容 Skill | 事实版本、引用和版权 |
| 定期清理知识库 | Wiki 体检、断链和重复检查 | 不自动删除或合并人工笔记 |
