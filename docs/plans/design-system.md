# 安居建业 WorkBuddy 蓝皮书设计系统

## 视觉基准

- 品牌来源：安居建业官网 `https://www.szajjy.com/`
- Logo：`docs/public/brand/anjian-logo.png`
- 背景：浅色主题使用真白色，深色主题使用近黑色。
- 主色：红 `#e21b22`、橙 `#f5831f`、黄 `#ffb800`。
- 正文：深灰 `#17191c`；弱化文字 `#666970`；分隔线 `#e4e1dc`。
- 字体：正文沿用系统中文无衬线字体；技术标签保留 Silkscreen 像素字体。

## 组件规则

- 导航只保留：首页、开始阅读、阅读指南、共创案例。
- 首页使用开放式列表、横向分区和清晰分隔线，不使用默认卡片堆叠。
- 红色用于主操作和关键标题，橙黄用于路径、编号和安全提示。
- 像素图标只承担导航与结构提示，不作为装饰性填充。
- 按钮、链接和图标必须具备键盘焦点、悬停和移动端触控状态。

## 首页可见文案锁定

- 标题：`安居建业 WorkBuddy 蓝皮书`
- 摘要：`从第一项真实工作，到可复用的 AI 工作系统。`
- 导航：`首页 / 开始阅读 / 阅读指南 / 共创案例`
- 按钮：`开始阅读 / 查看阅读路线`
- 四项价值：`真实任务 / 可复现 / 安全合规 / 系统沉淀`
- 六类任务：`公文与汇报 / 数据分析 / 会议与督办 / 制度与知识 / 项目研判 / 宣传与共创`
- 沉淀路径：`真实任务 / 实战案例 / 工作流 / 岗位应用`
- 共创标题：`这本蓝皮书，由安居建业的真实工作共同写成。`
