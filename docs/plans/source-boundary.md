# 内容来源与公开边界台账

> 本文件只用于私有编辑管理，不参与网站构建。原始材料保留在原位置，蓝皮书仓库不复制原文、截图、人员名单或业务数据。

## 使用级别

| 级别 | 定义 | 公开处理 |
|---|---|---|
| 允许具名 | 已确认可公开的产品名称、公开资料与通用能力 | 可以引用，但不得扩写内部事实 |
| 仅限方法抽象 | 内部实践可以证明方法有效，但项目本身不公开 | 只写通用任务、适用范围、步骤、验收和安全边界 |
| 不得使用 | 与安居建业无关、权属不清或包含敏感信息 | 不读取、不提炼、不进入正文 |

## 仅限方法抽象

| 私有来源 | 允许提炼的方法 | 公开时必须删除 |
|---|---|---|
| 安居建业 AI 协会周报及筛选流程 | 信息收集、筛选、复核、确认、发布、反馈 | 刊物名称、筛选记录、发布渠道、反馈数据 |
| 安居建业 PPT 模板 | 统一格式、内容检查、交付前复核 | 模板文件、真实页面、内部格式细节 |
| AI 协会门户、腾讯文档规划、群资料与培训共创 | 多人计划、任务台账、资料共享、活动组织 | 门户名称、群聊、人员、活动记录和内部链接 |
| 安居建业 AI 应用调查与分析看板 | 数据清洗、分组比较、证据展示、样本提示 | 部门、人员、样本、指标、真实结论和截图 |
| 安居建业制度清单与特殊不动产全链条资料 | 清单治理、文件比对、流程梳理、风险识别 | 制度目录、业务体系、项目数据和流程细节 |

## 不得使用

- 华彩、赛普、深业沙河及其相关项目、客户、文件和经验。
- 个人身份信息、联系方式、账号、密钥、访问令牌和本机凭证。
- 未公开经营数据、合同信息、项目金额、投资判断和审批意见。
- 未经批准的部门职责、组织架构、业务链条和公司正式结论。

## 公开前复核

1. 章节只引用通用方法编号，不引用私有来源名称。
2. 所有示例使用合成名称、合成数字和空白模板。
3. 事实性句子能追溯到公开资料或已确认的产品能力。
4. 没有依据的公司流程改写为“可采用的方法”，不得写成现状。
5. 公开构建产物通过敏感词、本机路径、人员数据和内部事项检查。
