# 安居建业 WorkBuddy 阅读指南：从入门到任务落地

这是一条面向安居建业新手、任务实践者和团队负责人的 WorkBuddy 学习路线。你不必从头背功能：先找到当前最接近的工作问题，完成一项可验收任务，再沿着基础使用、通用实战、进阶系统和任务落地逐步深入。

## WorkBuddy 新手入门路线

从[第一篇：先把 WorkBuddy 用起来](/bluebook/%E7%AC%AC%E4%B8%80%E7%AF%87%20%E4%BD%BF%E7%94%A8%E6%89%8B%E5%86%8C%EF%BC%9A%E5%85%88%E6%8A%8A%20WorkBuddy%20%E7%94%A8%E8%B5%B7%E6%9D%A5/)开始，依次完成：

1. 下载、安装和登录。
2. 认识主界面、任务与工作区。
3. 完成第一个可检查的任务。
4. 加载一个真正用得上的 Skill。
5. 再学习连接器、外部 API 与自动化任务。

建议边读边做，不要只浏览截图。每完成一章，至少留下一个可复用的提示词、模板或操作记录。

## 按真实任务选择 WorkBuddy 教程

直接进入[第二篇：案例篇](/bluebook/%E7%AC%AC%E4%BA%8C%E7%AF%87%20%E6%A1%88%E4%BE%8B%E7%AF%87%EF%BC%9A%E4%BB%8E%E4%B8%80%E9%A1%B9%E4%BB%BB%E5%8A%A1%E5%88%B0%E4%B8%80%E6%94%AF%20AI%20%E5%9B%A2%E9%98%9F/)，按问题选择章节：

- 处理 Word、Excel、PPT：第 11 章。
- 整理文件和版本：第 12 章。
- 安全远程协作与排障：第 13 章。
- 把模糊需求变成任务卡：第 14 章。
- 信息、知识和数据任务：第 15～17 章。
- 公文编制、会议与待办、制度与知识管理、项目研判分析：第 18～21 章。

案例跑通后，再回到第三篇，把你的做法整理为 Skill、自动化或多 Agent 系统。

## WorkBuddy 团队落地路线

重点阅读[第三篇：进阶系统](/bluebook/%E7%AC%AC%E4%B8%89%E7%AF%87%20%E8%BF%9B%E9%98%B6%E7%AF%87%EF%BC%9A%E6%8A%8A%E6%A1%88%E4%BE%8B%E5%8F%98%E6%88%90%E8%87%AA%E5%B7%B1%E7%9A%84%E5%B7%A5%E4%BD%9C%E7%B3%BB%E7%BB%9F/)与[第四篇：任务与成果落地](/bluebook/%E7%AC%AC%E5%9B%9B%E7%AF%87%20%E5%B2%97%E4%BD%8D%E4%B8%8E%E8%A1%8C%E4%B8%9A%E8%90%BD%E5%9C%B0/)。

团队实践建议同时记录：

- 输入资料和权限边界。
- 任务拆解方式和失败回退。
- 交付物的验收标准。
- 可以复用的 Skill、模板和自动化。
- 哪些步骤必须由人确认。
